# Type
- `Control + CMD + Space ` to type symbol

# VS code Short cut
- ⌘+⇧+P     select python interpreter
- ⌘+=       Zoom in 
- ⌘+-       Zoom out
- ⌘+,       Open Setting

# Add Autocompletion to VSCode's notebook
- Add below to `settings.json`
    ```json
    "python.languageServer": "Pylance",
    ```
